# general_four_state_fitness_function

```
ludics.fitness_functions.general_four_state_fitness_function(state, **kwargs)
```

Returns a general symbolic representation of the fitness of players in a state
with two players and two actions

### Parameters:

- `state`: _numpy.array_ - a length 2 binary vector

### Returns:

- _np.array_ - contains two _sympy.Function_ objects
