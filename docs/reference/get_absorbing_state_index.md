# get_absorbing_state_index

```
ludics.get_absorbing_state_index(state_space)
```

Returns the index(s) of absorbing states in the state space

### Parameters:

- `state_space` - _numpy.array_ - an array of possible states

### Returns:

- _numpy.array_ - the index of all all absorbing states in the state space.
  Returns _None_ if no absorbing states exist
